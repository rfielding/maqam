# Carpet target still

This is the visual target for the `carpet-guided-background` renderer after the midnight HUD/background pipeline was stabilized.

The successful mockup reads as a **woven maqam score artifact**, not a debug render:

- dark Persian-rug / manuscript field
- ornate border with corner medallions
- 3–4 adjacent organic territories, not rectangular cells
- territories share embroidered seams
- dense stitch ornament buries the underlying Hilbert/scaffold geometry
- ratio strings and rhythm groups appear as bead bands, tiny dots, hatching, and constellation motifs
- bright green monospace HUD remains readable on top
- palette: midnight navy, deep purple, green, teal, muted gold, dark red

## Rendering lesson

Hilbert should be hidden structure only:

```text
score/event stream -> Hilbert locality -> stitch direction / territory adjacency
```

The viewer should not see square Hilbert cells or a grid. The renderer needs to look like it actually **weaves** the image:

```text
base woven field
organic region underpaint
embroidered shared seams
dense ratio/rhythm stitch overlay
terminal HUD on top
```

## Prompt used for the target mockup

Create a concept mockup of the maqam player video frame as if the background were truly woven like a carpet. Image A is the target aesthetic reference: a dark, richly embroidered maqam carpet with organic shared territories, dense stitch ornament, beaded seams, jewel-tone regions, and a Persian-carpet / manuscript feel. Image B is the current video-frame screenshot showing the HUD layout and overall composition: a 16:9 video frame with a centered framed content area and bright green terminal-style text overlay. Generate a single 16:9 still frame that keeps the general screen composition from Image B but replaces the blocky background with a genuinely woven textile rendering inspired by Image A. The carpet should feel actually woven, with warp/weft texture, embroidered seams, subtle beaded rhythm bands, ratio-dot constellations, and dense hatch-like stitch motifs that bury any grid structure. Use 3-4 adjacent organic territories within a connected rug-like mass inside the border, not transparent blobs and not rectangular cells. Keep the palette dark and terminal-friendly: deep purples, greens, teals, muted golds, dark reds, midnight navy. Add an ornate rug border around the interior frame. Overlay a readable bright green monospace HUD similar to the current video frame in the upper left and across the top half, including lines for bpm, sustain, indexed sequence lines, and ratio strings, but keep it concise and legible rather than overly dense. The HUD should sit clearly on top of the carpet background. Overall feeling: mysterious, beautiful, impossible woven score artifact, highly detailed, richly textured, dark enough for readability, not a flat infographic, not a visible square grid.
